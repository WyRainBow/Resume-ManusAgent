## 📝 项目历史

基于 [OpenManus](https://github.com/FoundationAgents/OpenManus) 框架构建的智能简历优化 Agent，提供专业的简历分析、优化建议和对话式交互体验。

**核心功能**：
- 智能简历分析（模块化分析系统）
- 引导性对话（意图识别、主动引导）
- 现代化交互体验（流式输出、Markdown 渲染、打字机效果）
- 数据管理（简历数据存储、版本控制、对话历史）
